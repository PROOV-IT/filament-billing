# API documentation

`filament-billing` exposes a native Filament page that links to the Scramble documentation generated by `laravel-billing`.

## What it is for

- open the Scramble UI from the admin panel
- open the JSON schema for tooling or CI
- keep the billing API documentation discoverable from the same panel as the rest of the billing features

## Where it lives

- navigation label: `API docs`
- navigation group: the billing group configured in the plugin
- default slug: `billing/documentation`

## Paths

The page resolves the Scramble URLs from the billing package configuration:

- UI: `docs/api/billing`
- JSON: `docs/api/billing.json`

If the routes are registered, the page uses the named routes exposed by the core package. Otherwise it falls back to the configured paths.

## Recommended usage

1. Enable API docs in `laravel-billing`.
2. Visit the `API docs` page in `filament-billing`.
3. Open the Scramble UI or the JSON schema as needed.

## Notes

- The page does not duplicate the API docs.
- It only provides convenient access to the canonical Scramble output from the core package.
